# Gatito Petshop - Análise de Dados com Power BI

## Visão Geral do Projeto

Este projeto apresenta uma análise de dados de um petshop fictício, o Gatito Petshop, utilizando o Power BI. O objetivo é demonstrar a capacidade de transformar dados brutos em insights acionáveis, essenciais para a tomada de decisões estratégicas em um negócio. Como estudante de análise de dados, este projeto serve como um exemplo prático da aplicação de ferramentas de Business Intelligence para entender o desempenho de vendas e o perfil dos clientes.

## Principais Insights e Métricas

O dashboard do Gatito Petshop revela informações cruciais sobre o negócio:

*   **Faturamento Total:** 2,03 Milhões de Euros
*   **Média de Pets por Dono:** 2,61 pets
*   **Volume de Vendas:** 57 Mil unidades

Além disso, o dashboard detalha:

*   **Faturamento por Gênero:** Uma análise da distribuição do faturamento entre clientes do sexo feminino e masculino, indicando que 54,78% do faturamento vem de um gênero e 45,22% do outro.
*   **Faturamento por Bairro:** Os bairros de Itaquera (0,66 Mi€), Guaianases (0,63 Mi€), Cidade Tiradentes (0,39 Mi€), São Miguel Paulista (0,29 Mi€) e Vila Prudente (0,05 Mi€) são os que mais contribuem para o faturamento, destacando áreas de maior concentração de vendas.
*   **Faturamento pela Data:** Uma linha do tempo mostrando a evolução do faturamento ao longo do período analisado (19-12-2020 a 31-12-2022), com picos e vales que podem indicar sazonalidade ou impacto de campanhas específicas.
*   **Produtos:** Uma seção dedicada à visualização dos produtos vendidos, permitindo uma análise mais aprofundada do mix de produtos.

## Tecnologias Utilizadas

*   **Power BI:** Ferramenta principal para a criação do dashboard interativo e visualização dos dados.
